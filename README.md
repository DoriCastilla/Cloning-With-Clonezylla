# Cloning-With-Clonezylla
Cloning a Disk with Clonezilla
In this project, I changed the HD of my laptop and needed a clone of the old one to prevent the loss of my data. 

I did use: <br>
-	OS: Windows 11  <br>
-	D1: NVMe 250GB  <br>
-	D2: External HD 1TB  <br>
-	D3: NVMe 1TB  <br>
